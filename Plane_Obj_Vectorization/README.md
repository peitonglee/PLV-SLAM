## Getting Started

### environment 
cmake \
gcc  \
matplotlib_cpp (refer here: https://github.com/lava/matplotlib-cpp.git) \
python3 + matplotlib

### run
```
cd /path/to/PLV-SLAM/Plane_Obj_Vectorization
mkdir build && cd build
cmake ..
make
./Plane_Obj_Vectorization
```
